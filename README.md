## ðŸ› ï¸ Getting Started (Run Locally)

### Prerequisites
- [Ruby](https://www.ruby-lang.org/en/downloads/) (latest recommended)
- [Bundler](https://bundler.io/) (`gem install bundler`)

### Steps
1. **Install dependencies:**
	```bash
	bundle install
	```
2. **Serve the site locally:**
	```bash
	bundle exec jekyll serve
	```
3. Open your browser at [http://localhost:4000](http://localhost:4000)

> **Note:** If you change `_config.yml`, restart the server for changes to take effect.
